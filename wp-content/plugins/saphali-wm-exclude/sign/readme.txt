1.  Загрузите файл ключей на фтп.

2.  Отредактируйте файл WMSigner.ini:
	1-я строка -- Ваш_WMID;
	2-я -- пароль_к_резервной_копии_файла_ключей;
	3-я -- путь к файлу ключей (/home/my_site/wp-content/plugins/saphali-wm/sign/имя_файла_ключей.kwm ).

3. 	В FTP клиенте откройте свойства файла модуля WMSigner и установите для него права доступа 0700 (drwx------).
	В FTP клиенте откройте свойства файла WMSigner.ini и установите для него права доступа 0700 (drwx------). 

4. 	В случае неудачи скомпилируйте модуль WMSigner из исходников ( http://sourceforge.net/projects/wmsigner/ ) на той машине, где будет использоваться модуль ( подробнее здесь http://www.webmoney.ru/rus/developers/interfaces/xml/authdescr/wmsigner.shtml ).

5. Если совсем плохо - просто испльзуйте метод MD5.
